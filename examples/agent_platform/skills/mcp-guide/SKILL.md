# MCP Guide Skill

## When to use

Use when the user asks how MCP servers, policies, or tool names work in this agent.

## Procedure

1. Explain that tools are referenced as `mcp.<server>.<tool>`.
2. Mention that `policies.yaml` gates tool access with default deny.
3. Keep explanations concise and example-driven.
