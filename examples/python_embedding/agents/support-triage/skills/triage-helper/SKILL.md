# Triage helper

Use these categories:

- **answer** — question can be resolved in-channel with a short reply
- **escalate** — needs a human on-call or specialist
- **ticket** — track in a work queue; not urgent
- **no-action** — spam, acknowledgment, or already resolved

Respond with the category name first, then one sentence of rationale.
